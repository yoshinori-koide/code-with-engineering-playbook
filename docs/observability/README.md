# 可観測性

監視可能なシステムを構築することで、CSEの開発チームはアプリケーションの動作を測定できます。可観測性は次の目標を果たします。

- _アプリケーションの健全性_ の全体像を提供します。
- _顧客の業績_を測定するのに役立ちます。
- システムの _運用パフォーマンス_ を測定します。
- _障害を特定して診断_ し、問題を迅速に解決します。s
